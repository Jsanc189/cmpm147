Experiment 4 Infinite Worlds

Coral Planting Ocean Noise

Description

What are you trying to acomplish with this experiment?

In this experiment I partner coded with Raven Cruz. Together we worked on utilizing a tiles map with a noise value attached to each tile to create something like the reference photos above.

Technical

Describe in detail what you did technically and how it works?

The first thing we did was figure out how to add color to the tiles for an ocean floor like the one in the getty images. We did a few console.logs to see how to access number values to distinguish tiles for coloring. We also changed the tile width and height to match and also “zoom” us out to have a better look at how the noise function currently in the project affected the tiles.

Raven decided to add a 3rd dimension after we were messing around with the colors. This then made the “sand” colors move to mimic how water moves over the sand. The effect is quite mesmerizing!

Next, Raven and I downloaded an image pack from artist Pluto: https://paperpluto.itch.io/shore-things-pack. We made a lookup table for a small section of the items from the pack. In our placeTile function we created a new noise value based on the ti and tj coordinates and used that to make shells appear on the screen instead of placing a sphere in each box. We also set the size of the sprite with a constant variable. We also made the alpha of the colors of the tiles lower to make the sprites appear under the water.

I turned to chatGPT to help me figure out how to use the seed to generate different color palettes. I knew I needed to place an if else statement and replace the contents of the teals list and highlights list colors. I just wasn’t sure where to place it. GPT suggested the function p3_worldChangeKey. Of course as soon as that was shown I thought, “Well, of course!” Thus I added 2 color pallets that change the color palette based on the key.

Reflection

Each person that worked on the code should reflect on the process, the difficulties, and the successes of the experiment.

Raven and I split off from each other after this first, so this reflection will only have myself on here. Raven's will be on her own github.

Partner coding was a lot of fun, and it enabled us to bounce off of each other when someone got into a "I'm stuck" moment. With two heads working together on "what if we try this?" there was less time stalling, being unsure of what to do, and more work was accomplished because of it.

At the 4a start of this experiment, I didn't feel like we had too many difficulties. Again, this is largely due to the partner coding aspect. I will say, any obstacles didn't seem like they were because we turned to each other for ideas.
