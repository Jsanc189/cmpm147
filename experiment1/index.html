Experiment 1 - Fantasy Weapon Generator

Description

What are you trying to acomplish with this experiment?

In this experiment I am just re-familiarizing myself to HTML and JavaScript for making new content.

In this particular instance, I re-invented a generator to make fantasy type weapons for something like a Dungeons and Dragons type game.

Technical

Describe in detail what you did technically and how it works?

For this experiment I took a template provided by Wes Modes and re-invented its purpose. Utilizing the same Grammar style, I made a new grammar template and created several new lists of words or phrases to be used in the generation.

After playing with the new concept I jumped into the HTML side and reinvented the look of the page to make it look more fantasy like with changes to the fonts, colors and their gradients, framing of buttons and box elements etc.

Reflection

Each person that worked on the code should reflect on the process, the difficulties, and the successes of the experiment.

The process itself was a lot of fun. I liked coming up with a different application of generating text. The Process itself was just to build upon previous knowledge or to get ideas from ChaGPT.

The ideas utilized by ChatGPT were of text variety, or formatting ideas. I ended using them as starting points and then made it my own, much like Wes's original code was adapted.

That being said, because this was code being built upon, I did forget to realize I needed to call my main function when plugging it into my piece here. I spent a good hour trying to figure out where my error was, until I finally saw the commented code out on the bottom!
